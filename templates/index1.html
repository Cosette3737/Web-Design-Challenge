Diagnostic Breast Cancer
Prediction Tool

Using diagnostic features of cellular presentation
to predict histology type of breast biopsy tissue.




After collecting and analyzing data from the Breast Cancer Wisconsin Diagnostic Data set from Kaggle, a predictive diagnostic tool was created to determine if a biopsy should be classified as malignant or benign.


For this project we used:

Python | HTML | CSS | SQL | Flask | API | SKLearn | Jupyter Notebooks | Machine Learning | Microsoft Excel | SQLAlchemy Pandas | Matplotlib
